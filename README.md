# Pandas Data Analysis Projects

This repository contains exploratory data analysis (EDA) projects built using Python and Pandas to analyze business datasets.

## Projects

### 1. Retail Sales Analysis
- Analyzed sales performance and customer demographics
- Identified top-performing products and categories

### 2. Ecommerce Customer Behavior Analysis
- Customer segmentation using tenure buckets
- Spend distribution across marital status and order categories
- Stacked bar charts for category-wise spend composition

## Tools
- Python
- Pandas
